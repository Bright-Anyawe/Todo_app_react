 ## Installation


### Setup
1. Clone the repo:
   ```bash
   git clone git@github.com:Bright-Anyawe/Todo_app_react.git

2. cd Todo_app_react

3. npm install

4. npm run build - Builds the app for production.

5. npm run dev - Builds the app in development.


### Prerequisites
- Node.js
- npm
- Git
- Github
- React Testing Library(RTL)
